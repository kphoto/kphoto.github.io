title: Contact
---

The best place to reach the project is where it lives:

- **Issues and ideas** — [open an issue](https://github.com/kphoto/kphoto.github.io/issues)
  on the repository; build problems, typos and feature thoughts all welcome
- **Changes** — pull requests are open; the CI will run the same checks you
  can run locally with `./check.sh`
- **Email** — [hello@kphoto.example](mailto:hello@kphoto.example)

There is no comment system and no analytics on this site — by design. If a
post sparks something, the repository is the conversation.
