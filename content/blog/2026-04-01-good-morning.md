## Good morning, again

This post has **exactly the same title** as the one from March 22, and that is
the point. Post URLs on this site start with the publication date, so
`/blog/2026-04-01-good-morning/` and `/blog/2026-03-22-good-morning/` live side
by side without a fight.

The rule is simple:

- a title can repeat as often as you like
- a _file name_ — and therefore a URL — is unique per day
- the build fails loudly if a frontmatter date and a file name disagree

> One "good morning" per day is plenty anyway.
