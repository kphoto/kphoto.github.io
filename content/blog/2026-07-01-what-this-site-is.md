## The premise

kphoto is a working answer to one question: how far does the **modern web
platform** go with nothing added? The answer so far:

- a static site generator written in TypeScript 7, checked by the native compiler
- hand-rolled YAML, frontmatter and markdown parsers
- components scoped with declarative shadow DOM
- five themes, cross-document view transitions, and an Atom feed

## Built in the open

The [repository](https://github.com/kphoto/kphoto.github.io) holds every line:
source, unit tests, end-to-end tests and the CI that deploys this page. The
project is licensed AGPL-3.0-or-later and is developed with substantial
AI/LLM assistance — that disclosure sits in the README and in the footer of
every page, because it is part of the experiment.
